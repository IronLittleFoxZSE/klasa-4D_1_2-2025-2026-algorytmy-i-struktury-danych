# klasa-4D_1_2-2025-2026-algorytmy-i-struktury-danych

Zadanie 1:
Suma liczb od 1 do N
* Wczytaj liczbę N
* Oblicz sumę wszystkich liczb od 1 do N
* Wyświetl wynik

Zadanie 2: 
Silnia
* Wczytaj liczbę N
* Oblicz N! (silnia)
* Wyświetl wynik

Zadanie 3: 
Liczby parzyste z zakresu
* Wczytaj dwie liczby: początek i koniec zakresu
* Wyświetl wszystkie liczby parzyste z tego zakresu

Zadanie 4:
Odwracanie cyfr liczby
* Wczytaj liczbę (np. 1234)
* Wyświetl ją odwróconą (4321)

Zadanie 5:
Suma cyfr liczby
* Wczytaj liczbę
* Oblicz sumę jej cyfr
* Powtarzaj aż wynik będzie jednocyfrowy

Zadanie 6:
Kalkulator opłat parkingowych
* Wczytaj czas parkowania (w minutach)
* Pierwsza godzina: 4 zł
* Każda następna rozpoczęta godzina: 2 zł
* Oblicz koszt

Zadanie 7:
Bankomat
* Wczytaj kwotę do wypłaty
* Oblicz ile banknotów każdego nominału wydać (200, 100, 50, 20, 10)
* Minimalizuj liczbę banknotów
